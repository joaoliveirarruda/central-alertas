# central-alertas

Exercício integrador de Programação Orientada a Objetos: uma central de
monitoramento que gera, registra e exibe alertas de diferentes tipos.

O projeto integra os conceitos de interface pura, classe abstrata, herança,
`override`, `final`, o padrão Template Method e polimorfismo dinâmico via
`std::vector<std::unique_ptr<Base>>`.
